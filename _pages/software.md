## [rerand](https://github.com/xingjian-ma/rerand): R package for Rerandomization

`rerand` is an R package that provides functions for designing and analyzing experiments using rerandomization 
methodologies. It incorporates recent theoretical advances and offers user-friendly functionalities for researchers and 
practitioners.
